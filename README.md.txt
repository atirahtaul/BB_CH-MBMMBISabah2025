# 🎨 Custom HTML Looker Studio Page

This project is a custom-designed HTML page that mimics a Looker Studio layout but with interactive, clickable images. It's hosted for free using GitHub Pages.

## 📐 Features

- Fixed page size: **950px × 1795px**
- Custom **background image**
- Clickable images linking to:
  - Other HTML pages (in the same window)
  - PDF files
  - Google Forms
  - Google Drive file
  - WhatsApp chat links

## 🧩 Folder Structure

project-root/
│
├── index.html ← Main page
├── background.jpg ← Background image
│
├── img/ ← All clickable images
│ ├── link1.png
│ ├── form.png
│ ├── whatsapp1.png
│ └── ...
│
├── files/ ← Any downloadable PDF files
│ ├── file1.pdf
│ └── file2.pdf
